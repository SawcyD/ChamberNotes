# ChamberNotes

ChamberNotes is a modern, minimalistic note-taking application designed to help you organize and manage your notes efficiently. It features a sleek UI, customizable themes, version history, and more.

## Features

- Minimalistic and user-friendly interface
- Markdown support with live preview
- Customizable themes and accent colors
- Version history for notes
- Drag and drop functionality for organizing notes and folders
- Bidirectional linking of notes
- Graph view to visualize note connections
- Integration with external calendar applications

## Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/ChamberNotes.git
   cd ChamberNotes
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Run the application**
   ```sh
   npm run dev
   ```

## Usage

- **Creating a New Note**: Click the "New Note" button and enter a title. The note will be added to the current folder.
- **Creating a New Folder**: Click the "New Folder" button and enter a name. You can then drag and drop notes into the folder.
- **Version History**: Click the version history icon to view and restore previous versions of a note.
- **Customization**: Go to the settings panel to change the accent color and theme.
